# Google API の設定

Google Cloud Platform にいって、プロジェクト作成

- Google Calendar API
- Google+ API

を有効にする。(後者が有効じゃないとなぜかエラーでダメだった。Omniauthがデフォルトで求めるscopeの問題か？)

認証情報をダウンロードして .env_sample をリネームして書き込み、.env を作成してください。
